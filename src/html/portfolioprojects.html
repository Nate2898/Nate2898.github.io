RESTful API Notepad Website

I made a Notepad website that allows users to create, edit, and delete notes. It uses a RESTful API which allows users to create an account, login, and save notes to their account. The website is fully responsive and can be used on any device. The website is made using HTML, CSS, JavaScript, and NodeJS. The NodeJS is hosted on Render and uses MongoDB as the database. My NEFFEX website

NEFFEX, a popular music group, is known for creating copyright-free music and has released 100 original songs in 100 weeks. Their music has gained millions of fans worldwide, featuring hit singles like "Rumors" and "Fight Back," with over 250 million streams each. They've appeared on platforms like YouTube, Billboard, Forbes, ESPN, and Fox Sports, with a growing fan base and over 4 billion streams.

The website has a total of 28 pages that users can view and interact with. I've complied all of NEFFEX's songs into different genres, which has over a total of 200 songs with the music videos and social links of that song.

The website has undergone numerous revisions and transformations, eventually evolving into what it is today
