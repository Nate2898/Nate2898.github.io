Network Domain Project for InterTrode

In a team of 3 we created a comprehensive network domain design for InterTrode, a small fictional company. My roles involved setting up Migration, VLAN, Backup Solution, IPSec, and Domain Join Sophos, though I primarily focused on completing VLAN, Backup Solution, IPSec, and Sophos Firewall setup. VMware vSphere Network Infrastructure Project

In this project, I designed and implemented a virtualised network infrastructure using VMware vSphere. The project featured High Availability (HA) using pfSense firewalls, which were configured to provide failover for the network. I was the team leader, overseeing the implementation and ensuring all components worked seamlessly together. I also completed most of the work myself, as the other team member was away for a while. RESTful API Notepad Website

I made a Notepad website that allows users to create, edit, and delete notes. It uses a RESTful API which allows users to create an account, login, and save notes to their account. The website is fully responsive and can be used on any device. The website is made using HTML, CSS, JavaScript, and NodeJS. The NodeJS is hosted on Render and uses MongoDB as the database.
